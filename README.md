# hackweek

使用前，在merge文件夹下新建Config文件夹,Config文件夹里新建config.json,内容:
engine_port填程序监听的端口
database_config填数据库相关信息

```
{
    "engine_host": "",
    "engine_port": "",

    "database_config": {
        "driver": "",
        "user": "",
        "password": "",
        "host": "",
        "port": "",
        "db_name": "",
        "charset": ""
    }
}
```
